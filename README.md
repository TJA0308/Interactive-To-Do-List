# Basic Todo List

A small browser-based todo list app with localStorage persistence.

## Features

- Add tasks
- Mark tasks complete
- Filter all, active, or done tasks
- Delete individual tasks
- Clear completed tasks
- Saves tasks in the browser

## Run

Open `index.html` in a browser, or serve the folder with any static web server.

## Deploy

This repo includes a GitHub Pages workflow. Push the `main` branch to GitHub and the site will deploy through GitHub Actions.
